$mvvm-swiftui
========

$project is a Sample project to learn some concepts of swift-ui MVVM

Look how easy it is to use:

    import project
    # Get your stuff done
    project.do_stuff()

Covered topics
--------

- SwiftUI + MVVM
- Customized Views, use of @ViewBuilder
- Use of @MainActor, @StateObject, @Published variables
